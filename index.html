Quadro comparativo de linguagens



Linguagem Classificação Opinião
Javascript Tipagem Fraca Ela compõe a santíssima trindade, é essencial para o desenvolvimento web, e muito versátil, pois traz várias bibliotecas e frameworks o que facilita muito.
Java Tipagem Estáticas É uma linguagem versátil e confiável usada em várias de plataformas. Conhecida por sua portabilidade e segurança, o tempo passa, mas ela sempre permanece firme.
PHP Tipagem Dinâmica Comum para o desenvolvimento web do lado do servidor, embora tenha críticas em relação à qualidade do código. Ainda não tive contato com a linguagem.
Python Tipagem Forte É como o seu melhor amigo. Fácil de se dar bem, ajuda em tudo, desde automação até inteligência artificial. É uma linguagem que eu quero me aprofundar.
C# Tipagem Forte Desenvolvida pela Microsoft, é usada principalmente em aplicativos Windows e jogos com a Unity. Caso você esteja pensando em desenvolver algo para a plataforma Windows essa vai ser a linguagem certa.
C Tipagem Estáticas Uma linguagem de baixo nível usada em sistemas operacionais e programação de hardware. A mãe de todas as linguagens, muito forte, mas nada fácil de aprender pela complexidade dela.
